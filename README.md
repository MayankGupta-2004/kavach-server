# kavach-server

A Node.js/Express REST API server with structured public-facing interface, modular routing, and clean repository management.

## Tech Stack
- **Runtime:** Node.js
- **Framework:** Express.js  
- **Language:** JavaScript, HTML
- **Tools:** Git, .gitignore

## Features
- RESTful API design with modular route organization
- Public-facing static file serving
- Clean project structure with proper .gitignore configuration

## Getting Started
```bash
git clone https://github.com/MayankGupta-2004/kavach-server.git
cd kavach-server
npm install
node server.js
```

## Project Structure
```
kavach-server/
├── public/          # Static files
├── server.js        # Main server entry point
├── package.json     # Dependencies
└── .gitignore
```
